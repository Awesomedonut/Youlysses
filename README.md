# Youlysses

A Ulysses pact for you — a personal Chrome extension that tracks your screentime per website.

## Setup

1. Open `chrome://extensions` in Chrome
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked** and select this repo's folder
4. The Youlysses icon appears in your toolbar — click it to see your stats

## Features

- Tracks time spent per domain on your active tab
- Pauses when browser loses focus or you go idle
- Browse past days with arrow navigation
- All data stays local in `chrome.storage.local` — nothing leaves your machine
